# Micro-data
A part of microservices. That part give data from database and put data in it using GraphQL queries
* Ruby: ruby 3.0.0
* Installing
    * git clone <URL of project>
    * bundle install
* Database initialize:
    * rails db:create
    * rails db:migrate
* Run tests:
    * rspec
    * You can find text coverage open **coverage/index.html**
